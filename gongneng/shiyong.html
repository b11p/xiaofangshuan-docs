# 实用功能

# 汇率换算

将指定货币金额换算成人民币、美元、日元。如果是外币，同时按民生银行、兴业银行的汇率计算。

# 示例

汇率usd1

# 响应

CNY 6.52388
JPY 103.24272
CMBC CNY 6.5384000
CIB CNY 6.5418

# IP 归属地查询

查询 IP 归属地。IPv4 数据来自 IPIP在新窗口打开，IPv6 数据来自 ip.zxinc.org在新窗口打开。

# 示例

ip 2404:6800:400a:809::2004

# 响应

日本大阪府 Google Inc. 服务器网段
