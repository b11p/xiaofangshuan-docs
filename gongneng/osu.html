# 查看 PP-TTH 曲线

# 说明

查看自己的 PP-TTH 曲线。纵轴是 PP，横轴是 TTH。可以与其他人对比。

# 用法

pptth [vs 要对比的玩家] [, 模式]
1

其中，模式接受 0~3 的数字，和 std 等常见描述。

# 示例

pptth vs myangelzfxggg, 3
1

# 注意

数据来自白菜，必须使用过白菜才开始记录数据。

# 今日高光

# 说明

查询群内最近 24 小时 PP 及 TTH 增长最快的玩家。

# 使用方式

1

# 查询玩家信息

# 说明

查询玩家基本信息，例如 PP、游玩次数等。模式默认 std，可指定。

# 用法

查询自己的信息

~ [, 模式]
1

查询别人信息

where <用户名> [, 模式]
1

# 示例

~,3
1
where bleatingsheep
1

# 查询 PP+ 信息（仅限 std 模式）

# 说明

提示

由于 PP+ 要关门了，此功能可能不好用并且随时可能失效。

你可以使用消防栓查询 PP+在新窗口打开 上的个人信息。如果之前查询过此玩家，并且其数据当天有变化，则会显示增量。

# 使用方式

+
1
+<要查询的玩家>
1

# 示例

+
1
+myangelzfxggg
1
