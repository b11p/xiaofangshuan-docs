# 消防栓

消防栓本来是给 osu! 新人群开发的管理 bot，后来添加了许多其他功能，有与 osu! 有关的今日高光、pptth 等，也有与 osu! 无关的功能。

# 风控问题

消防栓现在加了接近 400 个群、超过 700 个好友，消息收发可能效频繁，容易被风控和封号，导致服务不可用。为了缓解此问题，建议您用自己的 QQ 号提供服务。见消防栓分身页面。
